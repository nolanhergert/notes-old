Aftertouch

A Python / OpenCV plugin that adds aftertouch/wind control capabilities to any standard MIDI keyboard. Only requires an external webcam and a simple 4-note initial setup.

Good demos would be for playing a stringed instrument (guitar/violin) and … adding some extra soul to a Moog piano or something.
